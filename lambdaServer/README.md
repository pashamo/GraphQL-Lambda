Running a GraphQL server instance from AWS Lambda

### More on the Stack ###

* GraphQL API
* Apollo Server
* AWS Lambda

The goal of this project is to understand serverless architecture and to build a serverless GraphQL API. By employing `apollo-server-lambda` package
